Google Sheets Template

Copy struktur kolom ini ke Google Sheets Anda (Row 1):

Project Name | Phase | Start Date | End Date | Progress | Description | Team Members | Child Tasks

Contoh Data (Row 2 dst):

Website Redesign | in-progress | 2025-01-01 | 2025-03-31 | 65 | Redesign company website | John Doe, Jane Smith | Design mockups; Develop frontend; Testing Mobile App | planning | 2025-02-01 | 2025-06-30 | 15 | New mobile application | Alice Brown, Charlie Wilson | Research; Wireframes; Prototype
Catatan Penting:
  • Phase: Harus planning, in-progress, atau completed
  • Date Format: YYYY-MM-DD (contoh: 2025-01-15)
  • Progress: Angka 0-100 tanpa simbol %
  • Team Members: Pisahkan dengan koma (,)
  • Child Tasks: Pisahkan dengan semicolon (;)
  1. Buka Google Sheets dan buat sheet baru
  2. Copy paste struktur kolom di atas ke Row 1
  3. Isi data project Anda mulai dari Row 2
  4. Klik File → Share → Publish to web
  5. Pilih format: Comma-separated values (.csv)
  6. Klik Publish dan copy URL yang muncul
  7. Paste URL tersebut ke form di bawah

Google Sheets Configuration

Tip: URL harus dalam format .../pub?output=csv atau .../export?format=csv
